# Pivoting-Lateral-Movement

## Objective

The main objective of this lab is to access and explore internal network systems that are not directly reachable by using pivoting techniques through a compromised host. This helps simulate real-world attack scenarios where attackers move through networks after initial access.

## Skills Learned

- Understand network segmentation and internal network structure.
- Perform internal network reconnaissance through a compromised system.
- Use pivoting techniques to access hidden or restricted networks.
- Configure and use Proxychains for traffic redirection.
- Use Ligolo-ng for secure tunneling and pivoting.
- Identify additional hosts and services within internal networks.
- Perform lateral movement between multiple machines in a lab environment.
- Maintain access while exploring deeper network layers.

## Tools Used

- Proxychains
- Ligolo-ng
- Kali Linux terminal
